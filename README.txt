-----------简介-----------
LightExam是一个开源的、轻量级的中小学生信息技术考试单机练习系统。
本软件致力于构建一个可扩展、易操作的考试平台，并可以简单的更改和编写题库。
本软件由xsun开发，采用Java(TM)语言编写，源代码托管在GitHub上，并通过Travis CI进行持续构建。
当前版本属于开发快照版，这说明本软件的主要特性还没有完成，可能会出现重大问题和程序崩溃。
请及时汇报问题。请勿将当前版本的软件应用于正式生产环境中！！

本软件的版本变更说明位于HISTORY.txt文件中。

---------安装与执行--------
1.安装Java(TM)8。使用浏览器打开http://java.com/zh_CN/download/manual.jsp，下载并安装适合您操作系统的版本。
2.下载并解压本程序的zip压缩文件。
3.双击执行LightExam.vbs
